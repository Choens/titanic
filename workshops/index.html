Icebergs Ahead! An introduction to R
This is the first workshop in the Titanic Introduction to R (TITR) series. If you know enough about R to be interested in this workshop, you are probably prepared to take it.
Participants learn primary data types, vectors and data.frames. Emphasis is placed on practical skills such as filtering data, and using a modern R IDE. By the the end of the workshop, participants will understand basic data management techniques, know how to run simple statistical tests, and create some basic plots of 2-dimensional data.
  • R-requisites: NONE
  • Stats-requisites: Particpants should understand the basics of Null Hypothesis Signifiance Testing (NHST).
Women and Children First! Regression in R
This is the second workshop in the TITR series. This assumes the participant has completed the first course or has at least a passing familiarity with R (or is an experienced programmer/fast learner).
Reviews and builds on the data management skills introduced in "Icebergs Ahead". Participants use these skills to build linear and logistic regression models.
  • R-requisites: Icebergs Ahead! or comparable experience using R
  • Stats-requisites: Participants should understand the concepts behind linear and logistic regression
